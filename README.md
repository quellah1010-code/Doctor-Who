# Doctor-Who Interactive Lab

Doctor Who / Rose door themed interactive web prototype.

This repo currently contains two React/Vite tools:

- **D&D Workbench**: a lightweight tabletop-style interactive narrative prototype for the Rose door scene.
- **Sound Lab**: a separate Web Audio API lab for testing short interaction sound effects before reusing them in the main workbench.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal. Use the top navigation to switch between **D&D Workbench** and **Sound Lab**.
